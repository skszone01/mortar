## ğŸ“ How to Use

### Finding Coordinates
1. Open in-game map (Press M)
2. Use the ruler to measure from the scale line at the bottom
3. Check X (Green) and Y (Red) coordinates at bottom right
4. Scale: 1mm = 10 units
5. Enter coordinates into calculator

### Target Presets
- **Click**: Load saved target
- **Right-click** (Desktop) / **Long press** (Mobile): Save current target
- **Clear button**: Reset all saved data

### Elevation Offset
Use offset buttons to adjust when:
- **Negative (-)**: Shell falls short
- **Positive (+)**: Shell overshoots
